# Rock Paper Scissors

A simple Rock Paper Scissors game built primarily with Javascript.  

## How to Play
1. Choose Rock, Paper, or Scissors by clicking a button.
2. The computer randomly selects its move.
3. A point is awarded to the winner of each round.
4. The first to reach 5 points wins the game.
